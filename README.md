# Sistema de Gestión de Permisos para Docentes

Sistema web para gestión de permisos y licencias de docentes escolares.

## Características

- ✅ Autenticación de usuarios por colegio
- ✅ Roles: Administrador y Usuario Colegio
- ✅ CRUD completo de docentes
- ✅ Gestión de permisos (vacaciones, enfermedad, capacitación, etc.)
- ✅ Filtrado por colegio
- ✅ Validación de fechas
- ✅ Interfaz responsive

## Tecnologías

- **Backend:** Python Flask
- **Base de datos:** PostgreSQL
- **ORM:** SQLAlchemy
- **Autenticación:** Flask-Login
- **Frontend:** HTML, CSS, JavaScript
